# 详细功能设计

## 1. 编辑器功能 (EditorPane)

### 核心
- [x] CodeMirror 6 集成，Vue 3 双向绑定
- [x] Markdown 语法高亮（`@codemirror/lang-markdown`）
- [x] 行号显示、活跃行高亮
- [x] 括号匹配、自动闭合
- [x] 代码折叠（标题/代码块）
- [x] 自动缩进（列表续缩进）
- [x] Tab 大小可配置（2/4/8）

### 搜索与替换
- [x] Ctrl+F 文件内搜索（`@codemirror/search`）
- [x] 搜索匹配高亮（编辑器内黄色背景）
- [x] 替换（Replace All / Replace One）
- [x] 大小写敏感、正则模式切换
- [x] 搜索词统计（n of m matches）

### 定位与导航
- [x] 跳转到行 (Ctrl+G)
- [x] 双击选中全文同词高亮
- [x] 光标位置实时显示（状态栏行/列）
- [x] 选中文本字符/词数统计

## 2. 预览功能 (PreviewPane)

### 核心
- [x] markdown-it 实时解析 → HTML 渲染
- [x] 编辑后 150ms 防抖更新
- [x] DOMPurify XSS 过滤
- [x] GitHub 风格 CSS（github-markdown-css）

### 格式支持
- [x] 标准 Markdown：标题/段落/粗斜体/删除线/列表
- [x] GFM：表格/任务列表/脚注/删除线/自动链接
- [x] 代码块 + highlight.js 语法高亮（180+ 语言）
- [x] 数学公式：行内 `$...$` + 块级 `$$...$$`（KaTeX）
- [x] Mermaid 图表：` ```mermaid ` 代码块渲染
- [x] 图片显示（本地路径/相对路径/DataURL）
- [x] 锚点生成（标题自动 id，用于 TOC 跳转）

### 同步滚动
- [x] 编辑区滚动 → 预览区按比例跟随
- [x] 点击 TOC → 编辑器定位到对应行
- [x] 搜索词预览区高亮（HTML 内 `<mark>` 包裹）
- [x] 滚动时当前章节指示器（预览区左侧标记）

### 视图模式
- [x] 双栏模式（编辑+预览并排）
- [x] 纯编辑模式（仅显示编辑器）
- [x] 纯预览模式（仅显示预览，阅读模式）
- [x] 预览宽度可拖拽调节

## 3. 工具栏 (Toolbar)

### 文字样式
| 按钮 | 快捷键 | 行为 |
|------|--------|------|
| 加粗 (B) | Ctrl+B | 包裹 `**text**` |
| 斜体 (I) | Ctrl+I | 包裹 `*text*` |
| 删除线 (S) | Ctrl+Shift+S | 包裹 `~~text~~` |
| 下划线 (U) | Ctrl+U | 包裹 `++text++` |
| 行内代码 | Ctrl+Shift+` | 包裹 `` `text` `` |

### 标题
| 按钮 | 行为 |
|------|------|
| H1 - H6 | 行首插入 `# ` ~ `###### ` |

### 段落
| 按钮 | 行为 |
|------|------|
| 引用 | 插入 `> ` |
| 水平线 | 插入 `\n---\n` |
| 换行 | 插入 `  \n` |

### 列表
| 按钮 | 行为 |
|------|------|
| 无序列表 | 插入 `- ` |
| 有序列表 | 插入 `1. ` |
| 任务列表 | 插入 `- [ ] ` |

### 插入
| 按钮 | 行为 |
|------|------|
| 链接 | 插入 `[text](url)` |
| 图片 | 弹出选择器 → 插入 `![alt](path)` |
| 表格 | 插入 3x3 表格模板 |
| 代码块 | 插入 `` ```language `` 模板 |
| 数学公式 | 插入 `$$ ... $$` 块模板 |
| Mermaid | 插入 `` ```mermaid `` 图表模板 |

### 操作
| 按钮 | 快捷键 | 行为 |
|------|--------|------|
| 撤销 | Ctrl+Z | CodeMirror undo() |
| 重做 | Ctrl+Shift+Z | CodeMirror redo() |
| 预览切换 | - | 循环: 双栏 → 编辑 → 预览 |

## 4. 多标签页 (TabBar)

### 功能
- [x] 新建标签（双击空白区域 / Ctrl+N）
- [x] 关闭标签（中间点击 / Ctrl+W）
- [x] 未保存标记（文件名旁圆点）
- [x] 拖拽排序（原生 HTML5 Drag & Drop）
- [x] 右键菜单：关闭、关闭其他、关闭右侧、复制文件路径
- [x] 标签溢出滚动（左右箭头 + 滚轮）
- [x] 文件修改外部检测（主进程 fs.watch → 提示重载）
- [x] 标签最大宽度 + 缩略文件名

### Tab 右键菜单
| 菜单项 | 行为 |
|--------|------|
| 关闭 | 关闭当前标签 |
| 关闭其他 | 关闭除当前外所有标签 |
| 关闭右侧 | 关闭当前右侧所有标签 |
| 复制路径 | 复制文件绝对路径到剪贴板 |
| 在资源管理器中显示 | 打开文件所在文件夹 |
| 固定标签 | 标签不可关闭（置顶 + 缩小） |

- [x] 右键菜单：新建文件/文件夹、重命名、删除
- [x] 文件拖拽移动（同盘移动）

### 功能

- [x] 树形结构显示（层级缩进）
- [x] 点击跳转到编辑器对应行

## 7. 搜索面板 (SearchPanel)

### 文件内搜索 (Ctrl+F)
- [x] 搜索输入 + 结果计数
- [x] 上一条/下一条导航
- [x] 替换 + 全部替换
- [x] 大小写/全词/正则选项

### 跨文件搜索 (Ctrl+Shift+F)
- [x] 输入查询 → 主进程递归搜索所有 .md 文件
- [x] 结果分组显示（按文件）
- [x] 每个结果显示：行号 + 行内容 + 匹配高亮
- [x] 点击结果 → 打开对应文件并定位到行
- [x] 搜索范围过滤（include/exclude 模式）
- [x] 搜索状态显示（搜索中... / 找到 N 个结果）

## 8. 自动保存 (AutoSave)

### 行为
- [x] 内容变化后 3 秒防抖自动保存
- [x] 已关联文件：静默写回原路径
- [x] 未关联文件：弹出另存为对话框
- [x] 状态栏实时显示保存状态
- [x] 编辑器关闭前检查未保存内容

### 状态指示
| 图标 | 含义 |
|------|------|
| ✅ 已保存 | 文件已同步到磁盘 |
| ⏳ 保存中... | 正在写入磁盘 |
| ⚠️ 未保存 | 内容已修改未写入 |
| 🔒 只读 | 文件无写入权限 |

## 9. 主题系统 (Theme)

### 模式
- [x] 跟随系统（监听 `nativeTheme` 变化）
- [x] 浅色模式（GitHub Light 风格）
- [x] 深色模式（GitHub Dark 风格）

### CSS 变量体系
```css
--bg-primary:         主背景（编辑器/预览）
--bg-secondary:       次背景（侧栏/底栏）
--bg-sidebar:         侧栏背景
--text-primary:       主文字色
--text-secondary:     次文字色
--text-muted:         弱化文字
--border-color:       边框
--accent-color:       accent 色（选中/链接）
--editor-font-family: 编辑器字体栈
--editor-font-size:   编辑器字号
--preview-font-family:预览字体栈
--preview-font-size:  预览字号
--line-height:        行高
--scrollbar-bg:       滚动条背景
--scrollbar-thumb:    滚动条滑块
--highlight-bg:       搜索高亮背景
--selection-bg:       选中背景
```

### 字体管理
- [x] 编辑器字体：预设 JetBrains Mono / Cascadia Code / Fira Code / Source Code Pro
- [x] 预览字体：预设系统默认 / Microsoft YaHei / PingFang SC / Noto Sans SC
- [x] 本地字体导入：`FontImportDialog` → 复制到 `assets/fonts/` → 注册 `@font-face`
- [x] 字号滑块：12px - 32px（步进 1px）
- [x] 行高滑块：1.2 - 2.0（步进 0.1）

## 11. 导出功能

### PDF 导出
- [x] 使用 Electron `webContents.printToPDF` 原生导出
- [x] 可配置：纸张大小（A4/Letter）、边距、打印背景
- [x] 含代码高亮 + 数学公式
- [x] 自动分页

### 图片导出
- [x] html2canvas 截取预览区
- [x] PNG 格式，可选缩放倍率（1x/2x/3x）
- [x] 只导出可见区域或完整内容

### ZIP 导出
- [x] jszip 打包整个项目
- [x] 可选：当前文件 / 选中文件 / 整个项目
- [x] 自动排除 node_modules / .git / dist
