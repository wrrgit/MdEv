### 第一章：微积分基础

微积分是研究函数的微分和积分的数学分支。微分描述函数的变化率，积分描述函数的累积。

**导数的定义**：函数 $f(x)$ 在点 $x_0$ 处的导数定义为：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

**定积分的定义**：函数 $f(x)$ 在区间 $[a, b]$ 上的定积分定义为：

$$\int_a^b f(x) dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i) \Delta x$$

### 第二章：线性代数

矩阵乘法：设 $A$ 是 $m \times n$ 矩阵，$B$ 是 $n \times p$ 矩阵，则 $C = AB$ 是 $m \times p$ 矩阵，其中：

$$c_{ij} = \sum_{k=1}^{n} a_{ik} b_{kj}$$

特征值与特征向量：对于方阵 $A$，若存在非零向量 $\mathbf{v}$ 和标量 $\lambda$ 使得 $A\mathbf{v} = \lambda \mathbf{v}$，则称 $\lambda$ 为特征值，$\mathbf{v}$ 为特征向量。

### 第三章：概率论

**贝叶斯定理**：

$$P(A|B) = \frac{P(B|A) P(A)}{P(B)}$$

**正态分布**：

$$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

**期望与方差**：

$$E[X] = \int_{-\infty}^{\infty} x f(x) dx, \quad \text{Var}(X) = E[(X - \mu)^2]$$

### 第四章：复变函数

**柯西-黎曼方程**：

$$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}$$

**柯西积分公式**：

$$f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z - z_0} dz$$
